# Backrooms Explorer

A liminal-space exploration website where every page is written as if it were a real explorer's journal.

Instead of explaining the Backrooms, the site presents them through field logs, photographs, environmental observations, and documented encounters. The goal is to make visitors feel like they are reading the notes of someone trapped inside an endless maze.

## Features

* Explorer journal entries
* Backrooms level archive
* High-quality Blender renders
* Environmental observations
* Threat level system
* Exploration status tracking
* Responsive design for desktop and mobile
* Dark, immersive interface inspired by liminal spaces

## Project Goal

The purpose of this project is to create a believable archive documenting the Backrooms through exploration rather than explanation. Every image and journal entry is presented as evidence collected during expeditions.
